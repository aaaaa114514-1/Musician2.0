**Musician 2.0** 是一款专为程序员和 Power User 打造的命令行音乐播放器。它不仅仅是一个播放器，更集成了网易云缓存解密、酷狗音乐下载、动态标签系统以及高度自定义的命令行交互体验。

## **1. 安装与初始化**

### **安装步骤**
1. 运行 `Musician_Installer.exe`。
2. **程序路径**：存放 `musician.exe` 的位置。
3. **BGM 数据路径**：存放音乐库、标签数据和配置文件的位置。
   * *注意：安装程序会自动处理 `settings.json` 中的路径关联。*

### **首次配置**
启动程序后，建议优先使用 `set` 指令配置网易云缓存路径：
`>> set netease_cache C:\Users\你的用户名\AppData\Local\Netease\CloudMusic\Cache\Cache`

## **2. 指令全索引**

### **通用指令**
| 指令 | 简写 | 功能 |
| :--- | :--- | :--- |
| `quit` / `exit` | `:q` | 安全退出程序（推荐方式）。 |
| `help` | `:h` | 显示所有可用指令。 |
| `set` | | 查看或修改配置项（如 `set volume 0.5`）。 |
| `common` | | 管理常用指令。支持在主界面通过 **上/下方向键** 快速选择。 |
| `history` | `:his` | 查看播放历史记录。 |
| `?` | `:?` | 显示当前播放歌曲的信息。 |

### **资源获取（下载与解密）**
#### **网易云音乐 (163)**
*   `check163` (`:163_cache`)：列出当前网易云桌面端的缓存歌曲。
*   `decode <编号>` (`:d <编号>`)：解密特定歌曲（支持范围，如 `decode 1-3 5`）。
*   `clear163` (`:163_clear`)：清空网易云缓存目录。

#### **酷狗音乐**
*   `search <歌名>` (`/s <歌名>`)：在酷狗音乐中搜索歌曲。
*   `download <编号>` (`/d <编号>`)：下载搜索结果中的歌曲。

#### **下载列表管理**
*   `savelist` (`:sl`)：显示本次运行下载/解密但尚未入库的歌曲。
*   `save <标签名>` (`:s <标签名>`)：将歌曲移入曲库并打上标签（如 `save 流行 摇滚`）。
*   `clear` (`:cl`)：清空当前下载列表。

### **播放控制**
*   `play` (`:p`)：播放/暂停。
*   `play <编号>`：根据当前列表编号播放。
*   `play <标签名>`：播放指定标签下的所有歌曲。
*   `stop` (`:st`)：停止播放。
*   `next` (`:n`) / `last` (`:prev`)：切歌。
*   `restart` (`:r`)：重新播放当前歌曲。
*   `volume <0-100>` (`:vol`)：设置音量。
*   `mode <s/c/r>` (`:m`)：设置模式（单曲循环/列表循环/随机播放）。

### **标签与曲库管理**
Musician 2.0 采用 **标签系统** 替代传统的文件夹分类。
*   `library` (`:lib`)：浏览整个本地 BGM 曲库。
*   `lookup <歌名>` (`:lu`)：在曲库中搜索歌曲。
*   `tag`：列出所有标签及对应的歌曲数量。
*   `tag <标签名>`：查看该标签下的所有歌曲。
*   `tag -a <标签名> <编号>`：为歌曲添加标签（如 `tag -a 健身 1-5`）。
*   `tag -d <标签名>`：从系统中删除该标签。
*   `tag -d <标签名> <编号>`：从特定歌曲上洗脱该标签。

## **3. 高阶特性**

### **指令连写 (Flags)**
`play` 指令现在支持带参数启动，大幅提升效率：
*   `play -m s`：切换到单曲循环并开始播放。
*   `play -v 40`：播放并将音量设为 40%。
*   `play -t 60`：播放并设置 60 分钟后自动停止。
*   *综合示例*：`play 1-10 -m r -v 30`（以 30% 音量随机播放编号 1-10 的歌曲）。

### **智能定时任务**
*   `timelimit <分钟>`：在指定分钟后停止。
*   `timelimit <HH:MM>`：在指定的时间点停止（如 `timelimit 23:30`）。

### **即时帮助系统**
在任何指令后添加 `-h` 即可查看其详细用法：
`>> tag -h`
